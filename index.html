About Me

I am a researcher bridging the gap between computer security and machine (and, deep) learning. My research focuses on enhancing the safety, security, and alignment of advanced AI systems, with a particular focus on developing defenses against adversarial attacks, as well as applying large language models to binary analysis and reverse engineering. I'm M.S. candidate at Sungkyunkwan University, advised by Hyungjoon (Kevin) Koo. I hold B.A. in software (from Sungkyunkwan University).

Research Interest

Adversarial Attacks and Defenses

This research focuses on developing robust defenses against adversarial attacks targeting AI systems. It emphasizes securing advanced architectures such as Retrieval-Augmented Generation (RAG) and self-reflective language models, as well as protocols like Model Context Protocols (MCP) and Agent-to-Agent (A2A) communication.

LLM-Guided Code & Binary Analysis

This work applies large language models (LLMs) to cybersecurity, focusing on enhancing binary analysis and reverse engineering. It targets challenges like Binary Code Similarity Detection (BCSD), function name recovery, and function type inference in stripped binaries—key techniques for understanding programs without source code or debugging symbols.
